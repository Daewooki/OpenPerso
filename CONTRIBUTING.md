# Contributing to OpenPerso

Thank you for your interest in contributing! Here's how you can help.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/OpenPerso.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Push and open a Pull Request

## Development Setup

See [README.md](README.md) for setup instructions.

## Guidelines

### Code Style

- **Backend (Python)**: Follow PEP 8. Use `ruff` for linting.
- **Frontend (TypeScript)**: Use ESLint + Prettier. Follow existing patterns.

### Commits

- Write clear, concise commit messages
- Use conventional commits when possible (e.g., `feat:`, `fix:`, `docs:`)

### Pull Requests

- Keep PRs focused and small
- Include a description of what changed and why
- Update documentation if needed

## Reporting Issues

- Use GitHub Issues
- Include steps to reproduce
- Include environment details (OS, Python/Node version)
